# 关键点标注操作指南

版本：V2.0

本文档面向标注员，说明如何在 Jiaolong Labeler 中完成关键点标注。

## 访问地址

- 本地模式：`http://127.0.0.1:3000/`
- 共享模式：`http://服务器IP:3000/`

请使用 `http://` 访问。

## 图片目录约定

- 待标注图片放在 `images` 下。
- 支持多级子文件夹（例如 `images/cam01/day/xxx.jpg`）。
- 页面中显示的图片名会包含相对路径，用于区分同名文件。

## 登录说明

- `shared` 共享模式：需要注册/登录。
- `local` 本地模式：无需登录，进入页面即可标注。

## 领取任务

进入标注页后点击：

```text
下一张（D）
```

可回到本次会话内上一张：

```text
上一张（A）
```

## 标注流程

每张图按以下流程进行：

1. 按左侧“标注流程”里显示的顺序点击角点。
2. 拖拽点位微调到准确位置。
3. 对遮挡点切换为不可见。
4. 点击“下一张”或“完成提交并下一张”自动保存。

注意：

- 角点数量和名称由模板配置决定，不固定为 4 个。
- 导出顺序由模板中的 `exportOrder` 决定。

## 鼠标与键盘操作

- 左键点空白：新增当前应标注的下一个角点
- 左键拖点：移动关键点
- 右键点某个点：切换该点可见性
- 右键拖拽：平移图片
- 鼠标滚轮：以鼠标位置为中心缩放

快捷键：

```text
A 或 ←    上一张
D 或 →    下一张
V         切换当前选中点可见性
```

## 可见性规则

- 可见：关键点在图像中可直接看见。
- 不可见：关键点被遮挡/模糊，但能根据结构推断大致位置。

建议：

- 可见点尽量精确贴合。
- 不可见点保持结构一致，不要随意偏移。

## 保存规则

系统会在以下操作时自动写入 `labels/*.txt`：

- 下一张（D）
- 上一张（A）
- 完成提交并下一张

若返回已标注图片，系统会优先读取已存在的 label，再次保存会覆盖旧文件。

## 质量检查建议

切图前请确认：

1. 必需角点均已标注（数量以页面提示为准）。
2. 点位位置准确，无遮挡误判。
3. 可见性设置正确。
4. 无明显顺序错误（顺序由页面流程提示和模板定义决定）。

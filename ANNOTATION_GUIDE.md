# 四关键点标注文档

版本：V1.0

本文档面向标注员，说明如何使用内网标注网站完成图片四关键点标注。

## 访问网站

在浏览器地址栏输入服务地址，例如：

```text
http://服务器IP:3000/
```

注意必须是 `http://`，不是 `https://`。

## 登录与注册

1. 第一次使用时，输入用户名和密码，点击“注册”。
2. 已注册用户输入用户名和密码，点击“登录”。
3. 每个标注员使用自己的账号，不要共用账号。

## 领取图片

登录后，点击顶部的：

```text
下一张（D）
```

系统会分配一张待标注图片。快捷键 `D` 或 `→` 也可以切换到下一张。

如果想回到当前会话中上一张图片，点击：

```text
上一张（A）
```

快捷键 `A` 或 `←` 也可以返回上一张。

## 标注总体流程

每张图片的标注流程如下：

1. 依次标出 4 个关键点。
2. 手动拖拽关键点微调位置。
3. 对不可见或被遮挡的点设置可见性。
4. 点击“下一张（D）”或“完成提交并下一张”，系统会自动保存 label 文件。

默认关键点顺序是：

```text
左上 → 左下 → 右下 → 右上
```

顺序错误会导致导出的 YOLO pose 关键点语义错误。

## 鼠标操作

### 标初始四点

左键按住图片上的目标位置，会出现局部放大框。移动鼠标精调位置，松开鼠标后落点。

### 移动关键点

左键拖动关键点即可移动。拖动时会出现局部放大框，放大框中有准星和当前点的位置。

### 移动图片

右键拖拽图片空白区域，可以平移图片。

### 缩放图片

鼠标滚轮可以缩放图片。放大后图像会显示原始像素，不做平滑模糊，便于精确标定。

### 切换可见性

右键点击某个关键点，可以切换该点可见性。

也可以先选中点，然后按快捷键：

```text
V
```

或点击左侧“切换可见”按钮。

## 可见性规则

每个点有两种状态：

```text
可见
不可见
```

请按以下规则判断：

- 如果关键点在图像中能直接看到，标为可见。
- 如果关键点被遮挡、模糊、反光覆盖，无法直接看到，但可以根据结构推断位置，标为不可见。
- 不可见点的位置允许有少量推断误差，但应保持几何一致，不要随意拖偏。
- 可见点应尽量精确对准真实位置。

## 保存规则

不需要手动保存草稿，也不需要手动导出。

当你执行以下操作时，系统会自动写入 label 文件：

- 点击“下一张（D）”
- 点击“上一张（A）”
- 点击“完成提交并下一张”

如果返回上一张图片修改，系统会读取之前生成的 label 文件。修改后再次切换图片，会覆盖旧 label 文件。

## 完成提交与下一张的区别

### 下一张（D）

当前图片会自动写入 label 文件，然后切换到下一张。适合正常连续标注。

### 完成提交并下一张

当前图片会自动写入 label 文件，并标记为完成，然后切换下一张。适合确认当前图片已经标注完毕。

## 快捷键

```text
A 或 ←    上一张
D 或 →    下一张
V         切换当前点可见性
鼠标滚轮  缩放图片
右键拖拽  移动图片
右键点    切换该点可见性
```

## 标注质量要求

1. 四个关键点顺序必须正确：左上、左下、右下、右上。
2. 可见关键点尽量精确贴到真实位置。
3. 不可见点保持结构一致，不要凭感觉大幅拖动。
4. 切换图片前确认四个关键点都已经标出，否则系统会提示不能保存。
5. 切换下一张前检查所有点的可见性，尤其不要漏掉不可见点。
